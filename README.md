# Обработчик заказов

Flask-приложение для управления заказами на электронику. Каталог товаров, оформление заказов, история, статистика.

## Запуск

pip install flask requests psutil
python app.py

Сервер откроется автоматически в браузере: http://127.0.0.1:5000

## API

GET / — главная страница
GET /api/products — список товаров (8 шт.)
GET /api/products/&lt;id&gt; — карточка товара
POST /api/order — создание заказа
GET /api/orders — история заказов
GET /api/stats — статистика

## Тестирование

python test_improved.py

35 тестов, 8 блоков по V-модели:

Блок 1. Системные — работоспособность целиком (10 тестов)
Блок 2. Валидация — отклонение плохих данных: невалидный email, отрицательная цена, XSS, SQL-инъекция (4 теста)
Блок 3. Верификация — расчёты, форматы, сохранение: формат ID заказа, формат даты, математика суммы, персистентность, согласованность статистики, идемпотентность (6 тестов)
Блок 4. Граничные — поведение на краях: 0 штук, 10000 штук, пустой заказ, имя 500 символов (4 теста)
Блок 5. Нагрузка — скорость: 10 заказов подряд, 5 параллельных заказов (2 теста)
Блок 6. Юзабилити — удобство API: формат JSON, понятность ошибок, технические заголовки (3 теста)
Блок 7. Производительность — память через psutil (RSS/VMS), RPS-нагрузка, рост памяти под нагрузкой, latency percentiles (p50/p95/p99), спайк-нагрузка 100 запросов (5 тестов)
Блок 8. Приёмочный — полный сценарий: каталог → товар → заказ → история → статистика (1 тест)

## Карточки товаров

8 товаров с полями: id, name, price, stock, emoji, images (3 фото с Unsplash), specs (8 характеристик: процессор, память, экран и т.д.).
